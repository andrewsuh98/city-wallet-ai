# City Wallet AI

An AI-powered city wallet that detects the most relevant local offer for a user in real time, generates it dynamically, and makes it redeemable through a simulated checkout. Built for the [Hack-Nation](https://hack-nation.ai/) x World Bank Youth Summit Global AI Hackathon, April 25-26, 2026.

This challenge is sponsored by **DSV-Gruppe** (Deutscher Sparkassenverlag), a company of the German Savings Banks Financial Group.
